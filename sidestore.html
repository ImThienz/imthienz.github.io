Hướng dẫn cài đặt và sử dụng SideStore



| Chuẩn bị:

  • Máy tính: Windows hay Mac tùy các bạn (Cài đặt sẵn AltServer trên máy, tham khảo tại đây).
  • Cáp kết nối iPhone với máy tính.
  • Thiết bị iOS/iPados 14.0 trở lên và yêu cầu có đặt mật khẩu máy.

|| Thực hiện:

1. Download tài nguyên
Trên iPhone/iPad
  • Tải ứng dụng WireGuard.
  • Tải file SideStore.conf

Sau khi tải xong file SideStore.conf về iPhone/iPad, tiến hành mở ứng dụng WireGuard sau đó bấm vào dấu cộng ở góc phải và tìm đến file SideStore.conf vừa tải về để thêm vào WireGuard sử dụng cho việc ký và gia hạn hạn ứng dụng.

Trên máy Mac/Windows
  • Tải SideStore iPA hoặc SideStore iPA (Beta).
  • Tải Jitterbug Pair: macOS hoặc Windows (x64 bit).
2. Pair Jitterbug
Đầu tiên là cắm iPhone/iPad vào máy Mac/Windows.
Pair Jitterbug: tiến hành giải nén file zip Jitterbug Pair đã download ở bước 1 sau đó mở Terminal (macOS) hoặc CMD (Windows) và chạy lệnh sau:

Đối với macOS: /Users/imthienz/Downloads/jitterbugpair
Trong đó imthienz chính là user máy của bạn. Cách đơn giản nhất là bạn kéo cái file jitterbugpair vô Terminal sau đó bấm Enter (return) và cấp quyền cho nó để nó có thể chạy.
Sau khi chạy xong nó sẽ sinh ra file *.mobiledevicepairing là được, nếu chưa sinh ra file này thì bạn chưa thực hiện thành công.


Đối với Windows:
  1. Kết nối iPhone của bạn với PC (Tin cậy thiết bị)
  2. Trên PC, tạo folder chứa file jitterbugpair-win64.zip và giải nén nó ra (Extract files)
  3. Tiếp đến, vào menu search trên windows (Windows + S) và tìm kiếm "Windows Security"
  4. Tại Windows Security, chọn menu Virus & threat protection và chọn mục Manage settings trong Virus & threat protection settings.
  5. Lướt xuống tại mục Exclusions, nhấp chọn mục Add or remove exclusions
  6. Chọn Add an exclusion và chọn Path chứa file jitterbugpair.exe (Ở đây mình đã add path SideStore, bên trong có chứa file jitterbugpair như ở bước 2)
  7. Vào folder jitterbugpair-win64, right-click và mở trong Terminal
  8. Đảm bảo iPhone đã kết nối với PC, kéo thả file jitterbugpair.exe vào CMD sau đó nhấn Enter, thấy menu cmd báo Success và xuất hiện file *.mobiledevicepairing là được.

Sau khi đã có file *.mobiledevicepairing, hãy đổi đuôi nó thành *.plist và chuyển file đó vào iPhone/iPad của bạn (bạn có thể tải lên Google Drive, Dropbox… sau đó trên iPhone/iPad truy cập vào link đó để tải về).

3. Cài đặt SideStore và sử dụng
Cài đặt:
Sử dụng AltServer (không phải AltStore) để cài đặt SideStore iPA:
  • Với macOS: bấm giữ phím Option + click chuột trái vào biểu tượng AltServer ở menu bar trên cùng của màn hình và chọn sideload .iPA
  • Với Windows: bấm giữ phím Shift + click chuột trái vào biểu tượng AltServer ở góc phải màn hình và chọn sideload iPA.

Sau khi cài đặt thành công và mở SideStore lên, nó sẽ yêu cầu file *.plist mà bạn đã tạo ra ở bước 2, lúc này hãy chọn đến file *.plist để hoàn thành quá trình cài đặt SideStore.


Sử dụng:
Đăng nhập Apple ID vào SideStore, khi cài đặt hoặc gia hạn ứng dụng hãy mở WireGuard lên và bật cấu hình SideStore.
P/s: nếu bạn đăng nhập Apple ID vào SideStore bị lỗi, hãy thử cách sau: Vào Cài đặt của iPhone/iPad -> Kéo xuống dưới bấm vào SideStore -> Ở phần Anisette URL nhập http://anisette.icert.cc/


Như vậy là mỗi khi cài ứng dụng hoặc gia hạn ứng dụng bạn sẽ không cần máy tính mà chỉ cần bật cấu hình SideStore bên trong ứng dụng WireGuard mà thôi.
